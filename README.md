# FridgeMatch

**Your personal chef that knows your fridge, your health, and your culture.**

## Features

- **Ingredient Selector** - 8 categories with emoji icons and search bar
- **Meal Match Calculator** - percentage based on available ingredients
- **Health Warnings** - Pregnancy (FDA/CDC 2024), Diabetic (ADA 2024), Heart Condition
- **Multilingual** - English, Arabic (RTL), French
- **SQLite Database** - meals and ingredients loaded from JSON on first launch
- **Favorites** - save and manage favorite meals
- **Shopping List** - auto-adds missing ingredients, checkable and deletable
- **Dietary Filters** - Halal, Vegetarian, Vegan, Gluten Free
- **Health Profile** - gender, age group, health conditions
- **AdMob Integration** - banner and interstitial ads

## Setup

1. Run `flutter pub get`
2. Run `flutter run`
